## 🆘 Need Help?

Common issues:
- **Images not showing?** → Check file paths match exactly
- **Styles not loading?** → Clear browser cache (Ctrl+Shift+R)
- **Map not working?** → You may need to enable Google Maps API
- **Contact form not working?** → Form needs backend/service (Formspree, Netlify Forms)
